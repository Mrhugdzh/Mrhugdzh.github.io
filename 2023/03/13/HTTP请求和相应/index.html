HTTP请求和相应

发表于 2023-03-13 阅读次数：

HTTP 请求格式

请求行

请求方法常用的有：

  • GET：地址栏访问、超链接访问一般都是 GET 方法，不安全，并且地址栏大小有限
  • POST：内容在请求体中，安全，没有长度限制

URL：请求的地址

版本协议：

  • HTTP/1.0：建立连接后立即断开，下次访问再次建立连接
  • HTTP/1.1：建立连接后可以不断开，待不在需要访问，再断开连接。

回车符换行符：\r\n

请求头部 ：

回车符换行符

一般用来分割请求头部和请求体

请求数据

一般是 POST 请求方式才有

HTTP 响应格式

状态行

状态码：

  • 200
  • 302：重定向
  • 304：从本地浏览器缓存拿取数据
  • 404
  • 500
